## 开发计划

- controller 需要支持 render
- 访问量/阅读量/浏览量：通过 log 文件定期收集(例如 10min)
- 前台访问用 cookie，后台 + 前台管理员访问接口带 token， token 每次进行校验

## 规范

### 返回接口

所有的接口返回的数据结构都使用 `{ code: number, message: string, data: any }`。
